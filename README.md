# plotly-image-viewer

Single page application to compare images produced by
the marvelous [plotly.js](https://github.com/plotly/plotly.js) library with the baselines.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
